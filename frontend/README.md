# Frontend Notes

This folder contains a static starter page for the Smart Laundry project.

- `index.html` is a basic landing page that links to the Flask login and signup pages.
- The real application UI is rendered using Flask templates in the `templates/` folder.
- Shared styles and scripts are located in `static/css/` and `static/js/`.

Use this folder as the place to expand frontend-only experiments, while keeping the Flask app inside `backend/`.
